# Tibia IP-Changer

Easy to use cross-platform ipchanger for Open Tibia with both GUI and Console interfaces.

### Build Instructions
In root directory:
```
mkdir build && cd build
cmake ..
make
```

### Required libraries
Qt5Widgets<br />
Boost System<br />
Boost Filesystem

```
sudo apt-get install qtbase5-dev libboost-system-dev libboost-filesystem-dev
```
