# Tibia IP-Changer

Easy to use cross-platform IP-Changer for Open Tibia written in C++. Tested for versions up to 10.98.<br />
Prebuilt binaries under releases

### Build Instructions
In root directory:
```
mkdir build && cd build
cmake ..
make
```

### Required libraries
Qt5Widgets

```
sudo apt-get install qtbase5-dev
```
